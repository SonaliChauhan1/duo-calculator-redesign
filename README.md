# Duo Group - Construction Cost Calculator Redesign

## Objective
Redesign the existing calculator with a modern, responsive interface while maintaining all required fields, CTA, and calculation explanation.

## Improvements
- Responsive layout using CSS Grid
- Improved visual hierarchy
- Enhanced accessibility
- Larger, more usable form controls
- Interactive estimate demonstration using JavaScript
- Modern card-based design aligned with financial services websites

## Technologies
- HTML5
- CSS3
- JavaScript (ES6)

## Prototype note
This is a front-end design prototype. The estimate is demonstrative only; a production version would connect these visual controls to Duo Tax’s existing calculation rules and report enquiry flow.
